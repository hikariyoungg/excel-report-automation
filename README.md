# EXCEL SIMPLE REPORT AUTOMATION

This project simulates a sales reporting system for NovaTech Supplies, a fictional technology hardware distribution company. The script processes Excel sales data and automatically generates summarized reports.

Este proyecto simula un sistema de informes de ventas para NovaTech Supplies, una empresa ficticia de distribución de hardware tecnológico. El script procesa datos de ventas de Excel y genera automáticamente informes resumidos.

## Features | Características del proyecto

- Reads Excel sales data
- Cleans missing values
- Calculates total revenue
- Generates automated reports
- Creates multiple report sheets

## Technologies | Tecnologías usadas

- Python
- Pandas
- OpenPyXL

## How to Run

1. Install dependencies

`pip install pandas openpyxl`

2. Run the script

`python report.py`
